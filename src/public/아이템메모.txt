1. 메인화면 - 메타마스크로 NFT 사용법?
    1. 설치
    2. polygon 지갑생성
    3. url 입력해 우리 서비스로 이동
    4. GNB 에 노출 된 지갑 주소를 복붙해 파티 참가 form 에 입력
2. 내 NFT : NFT 리스트
3. 내 NFT 상세 : NFT 이름, 파티명, 민터 지갑주소, 보유 기간 등등
4. 민팅 화면 : 엑셀파일로 올리기 (전화번호 & 이름 ? & 지갑주소), 이름은 래플 당첨자 발표 매칭에 사용??
    PWA 니깐 당첨자는 알람? -> 당첨자 지갑주소 보여주고 당첨자한테 알람 가기
5. GNB : 내 지갑주소 등의 정보


파티 참가 Form 입력 프로세스
1. Form 작성내용 중 'NFT & 경품이벤트 받을거니?' 항목에 yes 클릭한 경우 우리 서비스로 이동하는 주소 보여주면서 
지갑 생성하고 돌아와서 지갑주소 입력하라고 함
2. 구글 Form 입력 및 입금 기간 끝나면 액셀 뽑아서 민팅 화면에 가 엑셀파일로 올려 NFT 민트
3. 민트 완료 되면 PWA 를 이용해 알림 푸쉬
    알림 푸쉬는 어떻게 하는지(회원가입 필수 여부 알아야 함) 알아낸 후 디테일 적자.
4. PWA 및 푸쉬알람 기능 만들기 전엔, 직접 문자 메시지로 확인해보라 해야 됨
